# Volná místa na COVID-19 očkování

## Instalace

### Virtual environment

#### Create venv
`python3 -m venv venv`

#### Activate venv
`source venv/bin/activate`

#### Install requirements
`pip install -r requirements.txt`

### 
```
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
```
